Web ssh shell & sftp
--------------------
Just for fun!

Prerequisite
-------------------
* Java JDK 1.7 or greater
http://www.oracle.com/technetwork/java/javase/overview/index.html

* Tomcat server 7.0 or greater
https://tomcat.apache.org

* Browser with Web Socket support
http://caniuse.com/websockets
